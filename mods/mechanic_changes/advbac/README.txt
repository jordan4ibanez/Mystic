Advanced Bacteria Mod
=====================

Add 3 cells to your world which can mutated into something else:

-Simple cell --> The basis of everything.
-Blue cell   --> gives power to the cell to create thing. (Created with 4 Simple cell)
-Red cell    --> Dangerous but powerful cell.

--------------------

Version: 1.0
License: LGPL 2.1

Dependencies: default

Please report bugs on the forum topic.

Installation
--------------------------------------------------------------------

Unzip the archive, rename the folder to advbac and
place it in minetest/mods/minetest/

(  GNU/Linux: If you use a system-wide installation place
	it in ~/.minetest/mods/minetest/.  )

For further information or help see:
http://wiki.minetest.com/wiki/Installing_Mods
